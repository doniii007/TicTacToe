-------------------------
Tic Tac Toe using VRML
-------------------------

Description
-----------

A simple tic-tac-toe interactive game with 3D letters and grid.

Features
--------

- 3D letters and grid
- Background music
- Reset button


Tools used
----------

- VRML Pad 3.0
- Cortona 3D
